# ERP 产品成本管控系统交付文档

## 1. 文档目的

本文档用于说明 ERP 产品成本管控系统第一版的产品范围、业务规则、数据对象、页面结构、核算逻辑、版本对比逻辑和交付物清单。

当前阶段交付重点是产品需求确认与业务原型确认，不包含正式系统开发、ERP 对接和生产部署。

## 2. 产品定位

ERP 产品成本管控系统面向财务核算人员、产品成本会计和经营负责人，用于通过 BOM、物料采购价格和委外加工费自动计算产品成本，形成可追溯的成本版本，并支持不同版本之间的工段成本、物料成本和变化原因分析。

系统核心价值：

- 自动核算产品成本，减少手工 Excel 汇总工作。
- 按工段分类呈现产品成本结构。
- 按成本版本追踪 BOM、价格和委外费用变化。
- 自动识别新增物料、删除物料、用量变化、单价变化、委外加工费变化、工段变化和规格变化。
- 通过看板展示产品成本、版本差异、趋势和缺失价格风险。

## 3. 第一版范围

### 3.1 已纳入范围

- BOM Excel 上传。
- 一个 BOM 文件对应一个 SKU。
- 每次上传 BOM 生成一个产品成本版本。
- 物料价格 Excel 上传。
- 委外加工费上传。
- 价格匹配规则：物料编码 + 价格版本。
- 采购单价默认含税。
- 默认币种为人民币。
- 只核算末级物料，避免多层 BOM 重复计算。
- 支持产品 BOM 卷积计算验证。
- 按工段分类汇总成本。
- 当前版本默认对比同 SKU 最新上一版本。
- 支持用户手动选择两个或多个版本对比。
- 自动输出版本差异物料、差异金额、差异比例、差异类型和变更原因。
- 看板展示产品总成本、较上一版本差异、最大成本工段、最大上涨工段、缺失价格物料数。
- 支持 Excel 导出。

### 3.2 第一版暂不纳入范围

- ERP 系统实时对接。
- BU 字段展示和筛选。
- 复杂权限体系。
- 多币种汇率换算。
- 审批流。
- 多工序委外费用拆分。
- 自动供应商比价和采购寻源流程。

- 变更原因
- 单位

### 6.2 物料价格

关键字段：

- 物料编码
- 物料名称
- 供应商
- 采购单价
- 税率
- 币种

说明：

- 第一版委外加工费按一个物料的外部报价总成本处理。

- 较上一版本差异金额

## 7. 上传校验规则

| 校验项 | 校验规则 | 建议级别 |
|---|---|---|
| 缺失字段 | 必填字段为空 | 阻断 |
| 重复物料 | 同一个 SKU + 同一层级 + 子物料编码重复 | 警告或阻断，可配置 |
| BOM 用量异常 | BOM 用量为空、非数字、<=0 或异常偏大 | 阻断或警告 |
| 工段分类缺失 | 末级物料工段分类为空 | 阻断 |
| 价格缺失 | 末级物料按物料编码 + 价格版本无法匹配价格 | 阻断或允许生成并高亮 |
| 税率异常 | 税率不能识别为合理百分比 | 阻断 |
| 币种异常 | 第一版发现非 CNY 币种 | 阻断或警告 |

## 8. 成本核算规则

### 8.1 价格匹配规则

```text
匹配键 = 物料编码 + 价格版本
```

### 8.2 物料成本

采购单价默认含税。

```text
物料成本 = BOM用量 × 采购单价(含税)
```

### 8.3 不含税单价

```text
不含税单价 = 采购单价(含税) / (1 + 税率)
```

### 8.4 委外成本

```text
委外成本 = BOM用量 × 委外加工单价
```

### 8.5 单项总成本

```text
单项总成本 = 物料成本 + 委外成本
```

### 8.6 工段成本汇总

```text
工段成本 = SUM(同一 SKU + 同一成本版本 + 同一工段分类的末级物料总成本)
```

### 8.7 末级核算规则

```text
是否末级 = 是 的物料参与直接核算
是否末级 = 否 的组件不直接重复核算
```

## 9. 产品卷积计算规则

产品卷积计算用于验证多层 BOM 场景下，从末级物料向上滚算到组件和成品的成本是否一致。

### 9.1 末级物料节点

```text
末级节点成本 = 节点用量 × (采购单价(含税) + 委外加工单价)
```

### 9.2 非末级组件节点

```text
组件节点成本 = 组件用量 × SUM(直接子节点成本)
```

### 9.3 成品节点

```text
成品成本 = 成品用量 × SUM(一级组件节点成本)
```

### 9.4 验证要求

以下三项应保持一致：

- 成品卷积成本。
- 末级物料成本合计。
- 成本明细当前版本合计。

如果三者一致，说明系统未重复计算非末级组件。

## 10. 版本对比规则

默认对比逻辑：

```text
当前版本 vs 同一 SKU 下最新上一版本
```

用户也可以手动选择两个或多个版本进行对比。

| 差异类型 | 识别逻辑 |
|---|---|
| 新增物料 | 当前版本存在，上一版本不存在 |
| 删除物料 | 上一版本存在，当前版本不存在 |
| 用量变化 | 同一物料 BOM 用量变化 |
| 单价变化 | 同一物料采购单价含税变化 |
| 委外加工费变化 | 同一物料委外加工单价变化 |
| 工段变化 | 同一物料工段分类变化 |
| 规格变化 | 同一物料规格变化 |
| 价格缺失 | 当前版本末级物料未匹配到价格 |

输出字段：

- 物料编码
- 物料名称
- 当前版本成本
- 上一版本成本
- 差异金额
- 差异比例
- 差异类型
- 变更原因
- 差异说明

## 11. 工段多版本对比

看板应支持类似 `pivot check` 的横向版本对比视图：

| 工段分类 | 版本 1 | 版本 2 | 版本 3 | 差异列 | 差异说明 |
|---|---:|---:|---:|---:|---|
| 包装 | 成本 | 成本 | 成本 | 版本 3 - 版本 1 | 说明报价、BOM、风险 |
| 结构件 | 成本 | 成本 | 成本 | 版本 3 - 版本 1 | 说明二供导入、商务降本、核价变化 |
| PCBA | 成本 | 成本 | 成本 | 版本 3 - 版本 1 | 说明价格变化、测试结论、导入计划 |
| BOM 成本 | 合计 | 合计 | 合计 | 总差异 | 阶段总结 |

差异说明支持长文本，用于记录：

- BOM 未报价或预估成本风险。
- 二供报价。
- 商务降本。
- 核价更新。
- 供应商谈判。
- 打样测试进度。
- 下一步决策事项。

## 13. 验收口径

第一阶段验收建议检查：

- BOM 字段和价格字段是否符合业务要求。
- 单价含税口径是否已统一。
- 成本核算公式是否符合财务口径。
- 多层 BOM 卷积验证是否通过。
- 工段汇总是否能支撑经营分析。
- 版本对比差异类型是否完整。
- 差异说明是否能承载实际业务复盘。
- 页面结构是否覆盖上传、核算、对比、看板和导出闭环。
